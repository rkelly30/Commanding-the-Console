# Commanding the Console

## Tip and Tricks

- Making Commits:
    - Commit messages should be short and meaningfull
    - Make frequent commits
- VS Code Keyboard Shortcuts
    - **ctrl** + **`** for toggling the terminal window
    - **ctrl** + **k** followed by **s** to "Save All"
